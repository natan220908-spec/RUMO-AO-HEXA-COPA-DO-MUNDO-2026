# RUMO-AO-HEXA-COPA-DO-MUNDO-2026
Página interativa sobre a Copa do Mundo 2026, apresentando os jogos da Seleção Brasileira de forma organizada e visual. Inspirada em menus de jogos de futebol, permite acessar grupos e partidas com facilidade, oferecendo navegação simples, moderna e intuitiva ao usuário.
